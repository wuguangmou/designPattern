中介者模式：定义了一个中介对象，用来封装一系列对象的交互。
类型：行为型。中介者模式提供了一个中介类，该类通常处理不同类之间的通信，并支持松耦合，使代码易于维护。

适用场景：多各类相互耦合，形成了网状结构时。

优点：将类之间的一对多关系转化成一对一关系，降低了系统的复杂度；
      将各个类之间解耦。
缺点：中介者会庞大、变得复杂难以维护。
